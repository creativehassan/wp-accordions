WP Accordions with font color, background color styling options and 100% resposinve.

== Description ==

* WP Accordions is simple accordion plugin with rich featured options and templates, this plugin will allow you to design accordion that will be diplayed on your page or anywhere in the content with shortcode
* Shortcode inserter is new feature and enhance feature you can add and edit shortcode in editor by just one click and edit the shortcode by double clicking on placeholder.

<strong>Plugin Features</strong><br />

* Unlimited Accordions.
* Responsive Design.
* CSS Accordion (No JS)
* Use Via Editable Short-Codes.
* Accordion order sorting.
* WP Editor for accordion content add / edit.
* Heading Color.
* Heading Background Color.
* Container Text Color.
* Container Background Color.
* Container Text Align.
* Short-Codes support inside accordion content

<strong>Upcoming Features</strong><br />
* Hide any accordion without deleting it.
* Multiple Templates 
* Animated JQuery Accordion
* Font Awesome Icons(Add your own icons).
* Lazy Load.

== Installation ==

= Install from dashboard =
1. Visit 'Plugins > Add New'
2. Search for <strong>'WP Accordions'</strong>
3. Click on the 'Install Now'
4. Activate WP Accordions from your Plugins page.
5. You will see WP Accordions in admin menu.
 
= Manual upload =
1. Upload `/wp-accordions/` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently asked questions ==
= I can't find the setting screen. =
*  You will see WP Accordions in admin menu.

= How to add Short-code =
*  You will see a button <strong>'Accordion'</strong> near Add Media button, after clicking you will see screen to choose accordion you want to add,
You will see a placeholder image by double clicking that image you can edit the shortcode again.
